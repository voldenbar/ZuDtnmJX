# 项目简介

本系统名为基于YOLO11深度学习的交通标志与行人车辆检测系统，主要用途是在交通领域实现对交通标志、行人与车辆的实时检测。系统功能模块包括深度学习模型推理、图像处理和GUI界面展示。技术特点方面，采用YOLO11作为核心算法，结合Pyqt5构建了用户友好的GUI界面。该系统可处理1500多张标注图像，为交通管理和安全提供了有力支持。


## 核心功能
- 实时检测交通标志、行人与车辆
- 深度学习模型推理功能
- 图像处理模块
- GUI界面展示（基于Pyqt5构建）

## 适用场景
- 交通管理领域
- 交通安全监控
- 实时交通场景分析

## 技术特点
- 采用YOLO11作为核心算法
- 基于Python开发
- 集成Pyqt5界面框架
- 支持1500多张标注图像训练数据
